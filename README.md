# AIML-Run_LLMs_Locally_Ollama
This is how run LLMs locally using Ollama

(1) Install & Setup
Go to Ollama 
Download and Install to PC

(2) After Install
CMD --> ollama
Getting list ollama successfully installed

(3) Model Install and Use
Go to Ollama Site > Models

CMD --> ollama run qwen3.5 
(/bye - for stop)

Change LLM --> ollama run ModelName 

NOW CAN USE OLLAMA GUI
